# Choose Your SDK

Source: https://docs.cactuscompute.com/latest/docs/choose-sdk/

## Available SDKs

Cactus offers eight language bindings across different platforms:

- **React Native** (iOS, Android via npm)
- **Flutter** (iOS, Android, macOS via build)
- **Kotlin** (Android, iOS via build)
- **Swift** (iOS, macOS via build)
- **Python** (Arm Linux, macOS via build)
- **Rust** (All platforms via cargo)
- **CLI** (macOS, Linux via brew/source)
- **C++** (All platforms via header)

## Feature Support

All eight SDKs support:

- LLM inference
- Streaming responses
- Vision capabilities
- Gemma4 audio processing
- Transcription
- Diarization
- Function calling
- RAG / embeddings
- Cloud fallback functionality

## Quick Recommendations

**Mobile Development:**
- React Native — for existing React Native projects
- Flutter — for cross-platform needs
- Kotlin — native Android
- Swift — iOS/macOS with NPU acceleration

**Server / Scripting:**
- Python — inference and batch processing
- CLI — quick testing without coding

**Native Integration:**
- C++ — game engines, desktop applications
- Rust — systems-level integration with safe bindings
